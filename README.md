# Coladilla Portfolio

Personal developer portfolio built using React, TypeScript, and Tailwind CSS.  
Deployed via Vercel.

Live site:
https://coladilla-portfolio.vercel.app/

Tech Stack:
- React
- TypeScript
- Tailwind CSS
- Vercel

Overview:
This portfolio showcases my web development projects, including headless WordPress builds and fully coded React applications.

Features:
- Responsive layout
- Component-based structure
- Clean UI built with Tailwind
- Project showcase with external links

How to run locally:
1. npm install
2. npm run dev
